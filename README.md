## About
* **Language**: Kotlin
* **UI**: Jetpack Compose
* **Architecture:** MVI + UseCases
* **Reactive Code**: Kotlin Flow
* **Build Tool:** Gradle Kotlin DSL
### Dependencies
* **DI Framework:** Koin
* **Local storage:** Proto DataStore / Room
* **Navigation:** Jetpack Navigation Component
* **Network:** OkHttp3 / Retrofit2
* **Images:** Coil
* **QR Codes:** ZXing
